Working with Positions

Sticky

This is an example of an element with a sticky position. Its position is based on the user's scroll position.

Explanation: By using "position: sticky;" and "position: -webkit-sticky;", this box becomes sticky within its parent container and will stick to the top when scrolling.

Relative

This is an example of an element with a relative position. It moves based on the top, left, right, or bottom properties relative to its original position.

Explanation: The values "top: 120px;" and "left: 40px" moved the box 120 pixels down and 40 pixels right from its original position.

Fixed

This is an example of an element with a fixed position. It stays in a fixed position relative to the viewport, even when scrolling.

Explanation: With "top: 90px;" and "right: 10px;", the box remains in the same position relative to the viewport, regardless of scrolling.

Absolute

This is an example of an element with an absolute position. It is positioned relative to the nearest positioned ancestor element.

Explanation: Using "bottom: 20px;" and "right: 20px", the box is placed 20 pixels from the bottom and right edges of its containing element.
